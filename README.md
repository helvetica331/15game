Числовая головоломка "15"

How to use:
Игровое поле размером 4×4 содержит 15 плиток с числами от 1 до 15 и одну пустую клетку
  * Генерируется случайная конфигурация поля
  * Управление осуществляется с помощью команд (up, down, left, right и exit для выхода)
  * Работа программы завершается, когда все числа расположены в порядке возрастания (либо с помощью команды exit)
